Hamro Survey Application

This is a full stack application that helps the users to take a survey bu using the credits given to them.

Users can add more credits by the help of add credits button at the top.

The surveys will be send to the number of emails that the user has sent.

Survey response will be saved when the email recipients open the mail and click either yes or no button.

Technologies:
  • - React JS
  • - Node JS + Express
  • - MongoDB
  • - Sendgrid
  • - Ngrok
  • - Stripe for payment
  • - Google Login
