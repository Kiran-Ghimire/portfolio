Heart Disease Prediction System

This is a prediction system made by the help of machine learning.

Two algorithms i.e. Decision Tree and KNN is used.

The data is pre processed in the train test splits in the ratio of 0.7 to 0.3.

Data is properly visualized by the help of matplotlib.

Technologies:
  • - Python
  • - Numpy
  • - Decision Tree algorithm
  • - Pandas
  • - Matplotlib
  • - KNN algorithm
